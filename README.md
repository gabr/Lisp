# Lisp
Code based on tutorial from: www.buildyourownlisp.com

## Code

### MinGW
Code is in C compiled on Windows using MinGW.
Only two packages from mingw were used:
  - mingw32-base
  - msys-base

To install them the _mingw-get install_ command was used.
